# LeadPulse Frontend

Enterprise-grade Command Center for Lead Intelligence.

## Tech Stack
- **Framework:** Next.js 15
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Design:** Dark Mode / Command Center Vibe

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: Next.js App Router, contains the main page and layout.
- `src/components`: UI components.
- `src/lib`: Utility functions.

## Dashboard Features
- **Sidebar:** Navigation for Leads, Filters, and Intelligence.
- **Active Leads:** High-contrast table with intent signals.
- **Live Feed:** Real-time stream of detected signals (Mocked).
